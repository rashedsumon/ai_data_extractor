# AI-Powered Data Extractor

## Overview
This project is an AI-powered tool to:
- Extract data from Excel and PDF files (structured and semi-structured)
- Populate predefined Excel/Word templates
- Provide a user-friendly Streamlit front-end

## Features
- Multi-file upload (Excel/PDF)
- Table, text block, and key field extraction
- Handles scanned & digital PDFs
- Error logging and reporting
- Optional recurring task scheduling
